# Platform formats — Image Post Creator

## Instagram

- Caption: hook in first 125 chars; max ~2,200 chars
- Hashtags: 3–15 relevant; mix broad + niche
- Image: 4:5 portrait preferred for feed

## Facebook

- Caption: conversational; link in first comment if needed
- Image: 1:1 or 4:5; minimal text-on-image for reach

## LinkedIn

- Caption: professional hook + insight + question CTA
- Image: 1:1 clean; avoid heavy hashtag stacks (3–5 max)
