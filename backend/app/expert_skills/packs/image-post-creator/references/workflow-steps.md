# Workflow steps — Image Post Creator

## 1. Angle Research

- Restate goal, audience, platform
- 3–5 angles with: hook line, why it stops scroll, visual metaphor
- Pick recommended angle with rationale

## 2. Caption / Story Draft

**Thai default:** เรื่องเล่า 1 ตอน — ชื่อตอน + เปิดฉากผูกภาพ + 2–4 ย่อหน้า + คำถามปิดท้าย (see `thai-copy-style.md`)

**Classic caption** (only if buyer asks):

- Hook (first line visible before "...more")
- Body (2–5 short paragraphs)
- Soft CTA or question — avoid hard sell for story mode
- Alt text (natural Thai)
- 5–15 hashtags

## 3. Image Prompt

- Aspect ratio (1:1 or 4:5)
- Style block (photography, illustration, minimal, etc.)
- Scene description matching caption hook
- Text-on-image overlay spec if any
- Carousel table if multi-slide

## 4. Publish QA

- Caption length vs platform limit
- Hashtag count and mix
- Image prompt completeness
- Verdict: READY or NEEDS_CORRECTION
