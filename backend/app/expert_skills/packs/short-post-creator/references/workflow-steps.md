# Workflow steps — Short Post Creator

## 1. Research

- Restate goal, audience, platform
- 3 angle options with hook rationale
- Constraints: char limit, emoji, link policy

## 2. Draft

- 3 variants with different opening hooks
- Label each: Variant A / B / C
- Include character count per variant

## 3. Edit

- Pick best variant (or merge) as **Primary**
- One **Backup** variant
- Thread format if buyer asked (numbered posts)
- Final character counts

## 4. Publish QA

- Length vs platform limit
- Hook strength, CTA, no placeholder text
- Verdict: READY or NEEDS_CORRECTION
