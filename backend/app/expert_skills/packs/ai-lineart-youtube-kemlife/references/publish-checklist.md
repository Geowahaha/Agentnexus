# Pre-publish QA checklist

## Visual

- [ ] Every image is **16:9**
- [ ] Same MS Paint style across all shots (no drift to "pretty" art)
- [ ] Image changes match voiceover beats — no long static stretches

## Audio

- [ ] Voice clear, no long dead air
- [ ] Levels consistent; music low under voice

## Packaging

- [ ] Title creates curiosity (would **you** click without knowing the channel?)
- [ ] Thumbnail readable at phone size — one idea, bold text optional
- [ ] Length **8–13 minutes** unless buyer specified otherwise

## Files

- [ ] All images named by timestamp (`0-00`, `0-03`, …)
- [ ] Prompt table archived for the next video in the series

## QA verdict

- **READY** — all critical checks pass; buyer can render and upload.
- **NEEDS_CORRECTION** — list P0 fixes (wrong aspect, off-sync beats, weak hook) before publishing.
