# Marketplace deliverables — one run

## 1. Topic & Hook Research

- 3–5 video concepts with working titles (Thai or English per buyer)
- Why each title earns the click
- Suggested length and thumbnail hook words

## 2. Script + Timestamps

- Full voiceover script (8–13 min spoken pace unless specified)
- Timestamp column every 3–7 seconds on average
- Mark hook, re-hooks, and outro CTA

## 3. Shot Prompt Pack

- Table: timestamp | line | full MS Paint 16:9 prompt | filename
- Minimum shots = number of timestamps (typically 40–120 for 10 min)

## 4. Edit & Publish QA

- CapCut/DaVinci assembly steps (import order, sync tips)
- Title + thumbnail text options (3 variants)
- READY / NEEDS_CORRECTION verdict with checklist

**Pricing:** Standard OBOLLA content run ($0.99) + LLM tokens. Image rendering and video export are buyer-side costs.
