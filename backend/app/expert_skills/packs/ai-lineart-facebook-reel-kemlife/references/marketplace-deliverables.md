# Marketplace deliverables — one run

## 1. Hook & Angle Research

- 3–5 Reel concepts with scroll-stop hooks (Thai or English per buyer)
- Why each hook earns the first-second pause
- Suggested length (default 45–60s) and on-screen hook text

## 2. Script + Timestamps

- Full voiceover script (30–90s spoken pace unless specified)
- Timestamp column every **1–3 seconds**
- Mark hook (0:00), re-hooks, and optional part-2 tease

## 3. Shot Prompt Pack

- Table: timestamp | line | on-screen text | full MS Paint **9:16** prompt | filename
- Minimum shots = number of timestamps (typically 15–35 for 60s)

## 4. Reels Publish QA

- CapCut vertical assembly steps (import order, hook text overlay, sync tips)
- Facebook caption + hashtag block (copy-paste ready)
- READY / NEEDS_CORRECTION verdict with checklist

**Pricing:** Standard OBOLLA content run ($0.99) + LLM tokens. Image rendering and video export are buyer-side costs.
