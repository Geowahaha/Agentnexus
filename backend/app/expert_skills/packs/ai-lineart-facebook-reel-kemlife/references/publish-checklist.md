# Pre-publish QA checklist — Facebook Reels

## Visual

- [ ] Every image is **9:16 vertical**
- [ ] Same MS Paint style across all shots (no drift to "pretty" art)
- [ ] Image changes every **1–3 seconds** — no long static holds
- [ ] Hook text visible in **first 2 seconds** (burned-in or bold overlay)

## Audio & muted viewing

- [ ] Voice clear in first second (hook lands before scroll)
- [ ] On-screen keywords for muted viewers on key beats
- [ ] Music low under voice; no clipping

## Packaging (Facebook Reels)

- [ ] Caption opens with hook — first line works without watching video
- [ ] 3–8 relevant hashtags (mix Thai/English per buyer locale)
- [ ] Length **30–90 seconds** (sweet spot 45–60s unless specified)
- [ ] CTA optional: follow / part 2 tease in last 3 seconds

## Files

- [ ] All images named by timestamp (`0-00`, `0-02`, …)
- [ ] Prompt + caption table archived for next Reel in series

## QA verdict

- **READY** — all critical checks pass; buyer can render and upload to Facebook Reels.
- **NEEDS_CORRECTION** — list P0 fixes (wrong aspect, slow hook, weak scroll-stop, off-sync beats).
