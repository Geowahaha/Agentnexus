# SEO Expert Analysis — Marketplace Deliverables

Every paid run must deliver a **professional, downloadable-quality markdown report**.

## Required sections

### 1. Executive summary
- Overall SEO score (0–100) + 5 sub-scores
- Top 3 issues blocking growth
- Top 3 quick wins with highest ROI
- Competitive position in one paragraph

### 1b. Auto-discovered keywords
List keywords extracted from the live site crawl (not user-typed), with on-page evidence.

### 2. Competitor analysis
| Competitor | Domain | Strengths | Weaknesses | Your opportunity |
|---|---|---|---|---|
(3–5 rows, from public signals only — no backlink counts)

### 3. Content gap analysis
- Topics competitors cover that target site lacks
- Intent gaps (informational / commercial / local)
- Recommended new pages or content expansions (specific titles)

### 4. Technical SEO audit
Issues as:
```
[P0|P1|P2] Title — Category — Problem — Predicted impact — Fix — Verify
```
Categories: Crawlability, Indexability, On-page, Schema, Performance, Mobile

### 5. Core Web Vitals assessment
| Metric | Current | Target | Impact if fixed |
|---|---|---|---|
LCP, INP, CLS, TBT — use scan data; if unavailable say "verify via PSI"

### 6. Action plan
**Quick Wins (this week)** — numbered, owner-friendly
**Long-term (1–3 months)** — numbered, with effort estimate (S/M/L)

### 7. Predicted outcomes (conservative)
Table linking top 5 fixes → expected metric change → business impact range

## QA checklist (Report Generator)
1. Scores match scan evidence — no invented CWV numbers
2. Competitors are plausible for the industry/keywords
3. No backlink database claims or DA/DR numbers unless from scan
4. Every P0 has verification step
5. No secrets/tokens in output
6. Report is self-contained and client-ready
