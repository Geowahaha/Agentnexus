# On-Page SEO Rubric

## Title tags (/8)
- Unique per page, 50–60 chars ideal
- Primary keyword near front
- Brand suffix where appropriate
- No duplicate titles across site

## Meta descriptions (/6)
- Present on indexable pages
- 150–160 chars, includes CTA or value prop
- Unique per page

## Headings (/6)
- Single H1 per page, matches intent
- Logical H2→H3 hierarchy (no skips)
- Keywords natural, not stuffed

## Content quality (/10)
- Satisfies search intent for target query
- Answer-block structure for featured snippets
- Named authors/dates for E-E-A-T where relevant
- Internal links to related pages (≥3 on key pages)
- Image alt text on content images

## URL & linking (/5)
- Descriptive slugs, no random IDs on marketing pages
- Anchor text descriptive (not "click here")
- Orphan pages flagged

## Common failures (from production audits)
- Missing meta description (SEO score hit)
- H1 not first heading / multiple H1s
- Hero content hidden by JS animations (hurts crawl + CWV)
- Large JSON-LD in `<head>` blocking LCP
- Links without discernible names (accessibility + crawl)
