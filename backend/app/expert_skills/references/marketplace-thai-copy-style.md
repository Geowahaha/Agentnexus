# OBOLLA Marketplace — Thai copy style

## หัวข้อ skill (name) — บังคับ

รูปแบบ: **`{English product name} · {คำอธิบายไทยสั้นๆ}`**

- เก็บชื่อสินค้า/ศัพท์เทคนิคเป็นภาษาอังกฤษ — **ห้ามแปลทั้งก้อน**
- ส่วนหลัง `·` เป็นคำอธิบายภาษาไทยสั้น อ่านง่าย ไม่ใช่คำแปลตรงทุกคำ
- ตัวอย่างที่ถูก:
  - `Agent Flow · เอเจนต์โฟลว์`
  - `Image Post Creator · สร้างโพสต์รูปโซเชียล`
  - `Fable-5 Coding Agent Pro · โค้ดดิ้งเอเจนต์แบบ Pro`
- ตัวอย่างที่ผิด:
  - `โฟลว์เอเจนต์` (แปลทั้งชื่อ)
  - `สร้างโพสต์รูปโซเชียล` (ไม่มีชื่อ ENG)

## ศัพท์เทคนิค — เก็บภาษาอังกฤษ

ห้ามแปลหรือทับศัพท์เหล่านี้:

Agent Flow, Agent, QA, SEO, LLM, GPU, LoRA, Ollama, Grok, GPT, Gemini, Claude,
AIBotAuth, proof, scorecard, Fix Pack, robots.txt, llms.txt, pipeline, hook, CTA,
alt text, hashtag, timestamp, content gap, Core Web Vitals, Agent-Ready,
Facebook Reels, YouTube, Instagram, LinkedIn, Higgsfield, CapCut, MS Paint

## โทน (มุมกาแฟ OBOLLA)

- อธิบายง่าย เป็นกันเอง เหมือนเพื่อนแนะนำ skill ให้ฟัง
- บอกชัดว่า **หนึ่งรันได้อะไร** — ห้ามอวดเกินความสามารถจริง
- ประโยคสั้น–กลาง อ่านบนมือถือได้สบาย
- ไม่ใช้คำการตลาดหวือหวา ไม่สัญญาผลลัพธ์ที่วัดไม่ได้

## ภาษาไทยที่เป็นมนุษย์ (บังคับ)

**ทำ:**
- ประโยคสั้น สลับจังหวะ มีลมหายใจ
- คำไทยที่คนใช้จริง แต่ประโยคยังไทย
- ใช้ลูกศร `→` หรือชื่อขั้นตอน ENG สำหรับ pipeline

**ห้าม (แปลตรง / ราชการ):**
- "ในวันนี้เราจะมาพูดถึง…", "คุณเคยสงสัยไหมว่า…"
- คำเชื่อมซ้อน: "อย่างไรก็ตาม", "ดังนั้น", "ทั้งนี้", "นอกจากนี้"
- แปลศัพท์เทคนิคเป็นภาษาไทย: โฟลว์เอเจนต์, คิวเอ, จีพียู
- โทนรายงาน: "การศึกษาพบว่า", "มีความสำคัญอย่างยิ่ง"

## description

- 1–3 ประโยค เน้น deliverable จริง + ข้อจำกัด (เช่น ต้องมี GPU, รัน Higgsfield เอง)
- ราคาใส่ตามต้นฉบับ ($X/รัน หรือ ฟรี)
- อ้างเคสจริงได้ถ้ามีในต้นฉบับ (successcasting, pinpoint)
